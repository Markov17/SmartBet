This project leverages a combination of historical datasets and real-time APIs to calculate value bets and probabilities.

Historical Data (Past Results & Closing Odds)
Football-Data.co.uk

Description: The best free resource for comprehensive historical football data.

Details: Provides ready-to-use CSV files containing match results and odds from major bookmakers (Bet365, Pinnacle, etc.) dating back to the 90s.

Leagues: Covers Premier League, La Liga, Serie A, Bundesliga, Ligue 1, Primeira Liga, and more.

Access: Direct download, no registration required.

Live & Upcoming Odds (Real-time or Pre-match)
The Odds API

Free Tier: 500 requests per month.

Usage: Primary source for daily scripts to fetch current odds from multiple bookies.

Access: Requires email registration (no credit card needed).

API-Football / API-Sports

Free Tier: 100 requests per day.

Details: Provides fixtures, live scores, and pre-match odds.

Access: Simple email registration.

Statistics & Analytics (For Real Probability Calculation)
FBref.com

Data: Powered by Opta, offering advanced stats like xG (Expected Goals), player metrics, and team performance.

Access: Free downloads and manual scraping.

Understat.com

Data: Detailed xG data per match and per shot for major European leagues.

Access: Free web resource.
